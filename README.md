# Cheese-LinuxISO-SHA256-tools
## 简介
Cheese-LinuxISO-SHA256-tools是一款面向Linux系统的图形化工具，用于计算Linux ISO镜像文件的SHA256校验值，并与官方校验值一键比对，确保ISO文件完整、未被篡改。工具支持简体中文、英语、西班牙语三种语言切换，适配Linux系统原生主题与高DPI屏幕，操作简洁直观，兼容Debian、Ubuntu、CentOS、Arch等主流Linux发行版。

## 核心功能
1. 多语言支持：简体中文、英语、西班牙语界面自由切换
2. 多线程计算：分块读取ISO文件，计算过程不卡顿界面，实时显示进度
3. 一键比对：输入官方SHA256值，快速比对本地计算结果，结果直观提示
4. 系统适配：兼容主流Linux发行版，适配浅色/深色原生主题，支持高DPI屏幕
5. 全局启动：安装后通过终端单一命令即可启动，无需进入指定目录

## 安装步骤
1. 从Releases页面下载最新版本的tar.gz安装包：https://github.com/cfkbd2013/Cheese-LinuxISO-SHA256-tools/releases
2. 解压安装包：tar -zxvf cheese-linuxiso-sha256-tools-1.0.0.tar.gz
3. 进入解压目录：cd cheese-linuxiso-sha256-tools-1.0.0
4. 执行安装（自动安装PyQt5依赖）：sudo pip3 install .

## 使用方法
1. 安装完成后，任意终端输入命令启动工具：clst
2. 点击对应语言界面的“选择ISO文件”，导入需要校验的Linux镜像文件
3. 点击“计算ISO本地SHA256”，等待计算完成并获取文件校验值
4. 在“待对比SHA256值”输入框粘贴官方提供的校验值，点击“一键比对SHA256值”查看结果

## 卸载方法
终端执行命令：sudo pip3 uninstall cheese-linuxiso-sha256-tools -y

# Cheese-LinuxISO-SHA256-tools
## Introducción
Cheese-LinuxISO-SHA256-tools es una herramienta gráfica para sistemas Linux, diseñada para calcular el valor de comprobación SHA256 de los archivos de imagen ISO de Linux y compararlo con el valor oficial de comprobación en un solo clic, garantizando que el archivo ISO esté intacto y no alterado. La herramienta admite el cambio de interfaz entre chino simplificado, inglés y español, se adapta a los temas nativos y las pantallas de alta DPI del sistema Linux, con operación sencilla e intuitiva, compatible con distribuciones Linux principales como Debian, Ubuntu, CentOS y Arch.

## Características principales
1. Soporte multilingüe: cambio libre de interfaz entre chino simplificado, inglés y español
2. Cálculo multihilo: lee archivos ISO por bloques, el proceso de cálculo no ralentiza la interfaz y muestra el progreso en tiempo real
3. Comparación en un solo clic: ingresa el valor oficial de SHA256, compara rápidamente el resultado calculado localmente con indicaciones intuitivas
4. Adaptación al sistema: compatible con distribuciones Linux principales, se adapta a temas nativos claros/oscuros y admite pantallas de alta DPI
5. Inicio global: después de la instalación, se puede iniciar con un solo comando en la terminal sin acceder a un directorio específico

## Pasos de instalación
1. Descarga el paquete de instalación tar.gz de la última versión desde la página de Releases: https://github.com/cfkbd2013/Cheese-LinuxISO-SHA256-tools/releases
2. Extrae el paquete de instalación: tar -zxvf cheese-linuxiso-sha256-tools-1.0.0.tar.gz
3. Accede al directorio extraído: cd cheese-linuxiso-sha256-tools-1.0.0
4. Ejecuta la instalación (instala automáticamente las dependencias de PyQt5): sudo pip3 install .

## Modo de uso
1. Después de la instalación, ingresa el comando en cualquier terminal para iniciar la herramienta: clst
2. Haz clic en "Seleccionar archivo ISO" en la interfaz del idioma correspondiente e importa el archivo de imagen Linux a verificar
3. Haz clic en "Calcular SHA256 local del ISO", espera a que finalice el cálculo y obtén el valor de comprobación del archivo
4. Pega el valor de comprobación oficial en el cuadro de entrada "Valor de SHA256 a comparar" y haz clic en "Comparar valores de SHA256 en un solo clic" para ver el resultado

## Método de desinstalación
Ejecuta el comando en la terminal: sudo pip3 uninstall cheese-linuxiso-sha256-tools -y
